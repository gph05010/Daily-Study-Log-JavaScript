# Daily Study Log - JavaScript

JavaScript 기초 문법과 브라우저 기반 동작을 학습하며 정리한 실습 코드 저장소입니다.

교육 과정에서 진행한 HTML 기반 JavaScript 실습 파일을 모아두었으며, 변수와 연산자, 조건문, 반복문, 배열, 함수, 내장 객체, 타이머, DOM 조작 등 JavaScript의 기본 사용 흐름을 익히는 것을 목적으로 했습니다.

이 저장소는 JavaScript 기초 문법과 웹 페이지 내 동작 구현 방식을 학습한 기록입니다.

## 학습 목적

* JavaScript 기본 문법 이해
* 조건문, 반복문, 배열, 함수 등 기초 제어 흐름 실습
* HTML 문서 안에서 JavaScript를 실행하는 방식 학습
* 내장 객체와 타이머 함수 사용 흐름 이해
* DOM을 활용해 웹 페이지 요소를 선택하고 조작하는 방법 실습

## 주요 학습 내용

### 1. JavaScript 기본 문법

JavaScript의 기본 작성 방식과 브라우저에서 스크립트가 실행되는 흐름을 학습했습니다.

* HTML 문서 내 JavaScript 작성
* 변수와 값 처리
* 기본 입출력 흐름 이해
* 간단한 실습 예제 작성

### 2. 연산자와 조건문

산술 연산자와 삼항 연산자 등을 활용해 조건에 따라 다른 결과를 처리하는 방법을 실습했습니다.

* 산술 연산
* 비교 연산
* 삼항 연산자
* 조건에 따른 결과 출력

### 3. 반복문

while문과 for문을 활용해 반복 처리 흐름을 실습했습니다.

* while문 기본 구조
* for문 기본 구조
* 반복 조건 설정
* 반복문을 활용한 예제 작성

### 4. 배열과 함수

배열을 활용해 여러 데이터를 다루고, 함수를 통해 반복되는 코드를 분리하는 방법을 학습했습니다.

* 배열 생성 및 접근
* 배열 요소 활용
* 함수 선언 및 호출
* 매개변수와 반환값 흐름 이해

### 5. 내장 객체와 타이머

JavaScript에서 제공하는 내장 객체와 타이머 함수를 활용하는 방법을 실습했습니다.

* 내장 객체 사용
* 타이머 함수 활용
* 시간 흐름에 따른 동작 처리

### 6. DOM 조작

DOM을 활용해 HTML 요소를 선택하고, JavaScript로 웹 페이지의 내용을 변경하는 흐름을 학습했습니다.

* HTML 요소 선택
* 요소 내용 변경
* 이벤트와 동작 흐름 이해
* 웹 페이지 내 동적 처리 실습
